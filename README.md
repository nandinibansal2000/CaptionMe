### To run the model

 - Dowloand the pickle files from [this link](https://drive.google.com/drive/folders/1n6RN4HaWz36ei1Jt311e7eAb0-122lDe?usp=sharing)
 - Save an image on the root directory and name it 'Sample.jpg'
 - Run `python3 main.py`
